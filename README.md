# python-internship-task-5-
# Data Analysis on CSV Files using Pandas

## 📌 Project Overview

This project demonstrates how to analyze sales data from a CSV file using **Python**, **Pandas**, and **Matplotlib**. The program loads a dataset named `gadgets.csv`, performs data analysis, and visualizes the results using different charts.

---

## 🎯 Objective

* Load a CSV dataset using Pandas.
* Explore and analyze the dataset.
* Perform statistical analysis.
* Group and summarize sales data.
* Filter records based on conditions.
* Visualize data using charts.

---

## 🛠️ Technologies Used

* Python 3.x
* Pandas
* Matplotlib
* Jupyter Notebook / Google Colab / VS Code

---

## 📊 Dataset

The dataset (`gadgets.csv`) contains sales information with the following columns:

* OrderID
* Product
* Category
* Region
* Sales
* Quantity

## ✨ Features

* Load CSV file using Pandas
* Display the first five rows of the dataset
* Show dataset information
* Generate summary statistics
* Display the shape of the DataFrame
* Check for missing values
* Calculate total sales
* Calculate total quantity sold
* Group sales by category
* Group sales by region
* Group sales by product
* Calculate average sales by category
* Find maximum and minimum sales
* Identify the highest-selling product
* Filter products with sales greater than 20,000
* Filter products belonging to the Electronics category
* Sort products based on sales
* Create multiple charts for visualization

The project generates the following charts:

* 📊 Bar Chart – Sales by Category

## ▶️ How to Run

### 1. Install the required libraries

```bash
pip install pandas matplotlib
```

### 2. Place the dataset

Save `gadgets.csv` in the same folder as `data_analysis.py`.

### 3. Run the program

```bash
python data_analysis.py
```

Or open the script in **Jupyter Notebook** or **Google Colab** and execute all cells.
